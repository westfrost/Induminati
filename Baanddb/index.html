⬆ GITHUB SYNC — PUSH / PULL DATABASE
ÅBEN ▾

Database: westfrost/Induminati → Baanddb/database.json  ·  Pull fra GitHub virker uden adgangskode.

Sådan virker det: Alle der kender jeres fælles adgangskode kan pushe — I skal ikke oprette et nyt GitHub-token på hver ny pc. Adgangskoden låser et delt, krypteret token op, som ligger i selve repo'et.

⚠️ Repo'et er offentligt, så den krypterede fil kan ses af alle — sikkerheden afhænger derfor alene af hvor svær adgangskoden er at gætte. Vælg en lang adgangskode og del den kun med dem der skal kunne pushe.
OPSÆT ELLER SKIFT DELT TOKEN

Kør dette én gang, eller når adgangskoden skal skiftes. Kræver et rigtigt GitHub-token denne ene gang — herefter kan alle bruge adgangskoden i stedet.

1. Gå til github.com/settings/tokens/new
2. Sæt navn (f.eks. "baanddb"), vælg Contents: Read & Write under repository permissions
3. Generer token og indsæt det ovenfor sammen med den nye adgangskode
HURTIG-TAST — MANGE POSTER PÅ ÉN GANG

Tab/Enter springer til næste felt. Når en række er udfyldt springer Enter automatisk til næste række. Tryk Gem alle rækker når du er færdig.
